## Requirements

- iOS 14.5+ / macOS 11.2.3 
- Xcode 12.5+
- Swift 5.3+

## Installation:

Install the dependencies

```sh
1- cd NYTimesHeadlines
2- pod install
3- open -a Xcode NYTimesHeadlines.xcworkspace
```
